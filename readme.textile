*Como exportar o script de um banco de dados SQL Server 2008 R2 para o Windows Azure SQL?*
O Windows Azure SQL não é igual ao SQL Server 2008 R2, sendo assim, ao exportar um script do 2008 R2, é necessário modificar um parâmetro para que o script possa ser executado no Azure. Siga os passos:
# Botão direito no banco de dados, Tasks, Generate Scripts...
# Ao invés de escolher "Script entire database", escolha "Select specific database objects" e marque a opção "Tables" (todas). Isso é porque o banco de dados Cerebello já vai existir no servidor porque é melhor que ele seja criado através da ferramenta.
# Em "Set Scripting Options", selecione "Advanced"
# Em "Script for database engine type", escolha "SQL Azure Database"
# Execute o Script no Windows Azure
